# CyberShield
CyberShield is an AI-powered cybersecurity checklist web app that evaluates user security habits, generates a risk score, and provides personalized recommendations. It features attack simulations, a modern dashboard UI, and helps users improve digital safety through interactive insights.




# 🧠 CyberShield – Smart Cybersecurity Checklist

🚀 An AI-powered web application that analyzes user cybersecurity habits, calculates a security score, and provides personalized recommendations to improve digital safety.

---

## 🌟 Features

- 🔐 User Authentication (Login/Signup)
- 📋 Smart Cybersecurity Checklist
- 📊 Security Score (0–100)
- ⚠️ Risk Level Detection (Low / Medium / High)
- 🤖 AI-Powered Suggestions
- 🧠 “What If Attacker” Simulation
- 📈 Interactive Dashboard UI
- 💾 Data Storage with Firebase

---

## 🖥️ Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Firebase (Auth + Firestore)
- AI Integration: OpenRouter API
- Deployment: Firebase Hosting

---

## ⚙️ How It Works

1. User logs in to the platform  
2. Fills cybersecurity checklist  
3. System calculates security score  
4. Risk level is assigned  
5. AI generates personalized suggestions  
6. Results are stored and displayed in dashboard  

---

## 📊 Scoring Logic

- MFA enabled → +30  
- No password reuse → +20  
- Regular updates → +20  
- Avoid public WiFi → +30  

**Risk Levels:**
- 80–100 → Low Risk  
- 50–79 → Medium Risk  
- Below 50 → High Risk
